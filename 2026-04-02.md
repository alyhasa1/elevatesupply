# 2026-04-02 Changelog

## Changes Made
1. **Hero Section Updates (`src/pages/storefront/Home.tsx`)**:
   - Increased the minimum height of the hero section from `min-h-[50vh]` to `min-h-[60vh]` (approx 10% increase).
   - Added `max-w-7xl` to the hero's inner container to perfectly align the text with the top navigation boundaries.

2. **Streamlined Operations Section (`src/pages/storefront/Home.tsx`)**:
   - Converted the section from a dark theme (`bg-stone-950 text-white`) to a light theme (`bg-white text-stone-900`).
   - Adjusted paragraph text to `text-stone-600` for better readability.
   - Updated the step cards to have white backgrounds, lighter borders (`border-stone-200`), and a softer shadow to match the light theme.
   - Added blurred abstract glowing background elements in `stone-200` and `orange-400` to bring subtle vibrancy to the section.
   - Updated the connecting line to a `stone-200` gradient and changed the circular icon containers to use `stone-200` borders and `stone-400` icons that turn `orange-600` on hover, along with scaling animations.

3. **Header Logo Text (`src/layouts/StorefrontLayout.tsx`)**:
   - Added the text "Elevate Supply" next to the main header logo image.
   - Formatted the text to match the brand colors, using `text-sky-950` for "Elevate" and `text-orange-600` for "Supply", which turns `text-orange-600` on hover.
   - Implemented a two-tone font weight design: "Elevate" is `font-[600]` while "Supply" is `font-[300]`, matching the elegance of the reference image.
   - Imported and applied the **Outfit** font family globally and specifically for the logo text to give it a wider, more elegant, and premium geometric feel.
   - Applied tight letter spacing (`tracking-tight`) and reduced word spacing (`ml-[-2px]`) for a unified logo appearance.
   - Set the text to hide on very small screens (`hidden sm:block`) to preserve mobile responsiveness.
   - Replicated the identical styling in the Footer logo area.

4. **Theme and Brand Colors**:
   - Replaced all instances of `rose` theme colors with `orange` across the application (components, pages, index.css) to align with Elevate Supply's brand.
   - Replaced all `brand` and `secondary` palette variables in `src/index.css` to use orange and sky blue variants.
